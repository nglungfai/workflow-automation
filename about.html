About this site

Maintained by the Product team

This site is a working knowledge base for using Claude Cowork at Predactiv — practical tutorials, workflow write-ups, and prompting tips aimed at helping the team get real value out of AI tooling during our transformation.

Who this is for

Product managers, data analysts, data scientists, ML/data engineers, and anyone at Predactiv curious about applying Cowork to their daily work.

Contributing

This site lives in a public GitHub repo and is built as plain HTML/CSS, published via GitHub Pages. To add a post:

  1. Copy an existing file in /posts as a template.
  2. Write your tutorial or tip in the same structure (title, meta line, sections).
  3. Add a card for it to index.html.
  4. Open a pull request.

Have a workflow to share?

If you've built a Cowork workflow worth documenting, reach out to the Product team — we'll help you turn it into a post.
